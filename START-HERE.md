## 🆘 If Something Goes Wrong

### "MongooseServerSelectionError"
→ You didn't whitelist your IP or didn't wait 2 minutes
→ Go back to Step 1

### "Invalid credentials"
→ Make sure you're typing: admin@seemee.com / admin123
→ No spaces, no typos

### "401 error" when uploading
→ You're not logged in
→ Go to Step 3 and login first

### "EADDRINUSE" error
→ Backend is already running
→ That's OK! Skip to Step 3
